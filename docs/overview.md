## Summary
RecipeGenius is a personal recipe book app that lets users save and organize recipes, import from the web with auto-extracted ingredients/steps/images, use AI to normalize ingredients and units, search/filter by tags/ingredients/cuisine/time, and plan weekly meals with one-click shopping lists.

## Goals
- Organize and search personal recipes efficiently
- Import recipes from the web reliably
- Plan weekly meals and generate shopping lists quickly

## Target Audience
- Primary: Individual home cooks
- Secondary: Families sharing a household

## MVP Scope
- Smart Web Import
- AI Ingredient Normalization
- Search & Filters
- Meal Planner + Shopping List
- User Authentication

## Non-Goals (MVP)
- Public recipe sharing/community
- Advanced nutrition/macros
- Mobile apps (web-first)

## Success Criteria
- Import a recipe from a popular site in < 5s with an editable preview
- Search returns relevant results within 300ms (post-index)
- Generate a weekly plan and deduplicated shopping list in under 1 minute
